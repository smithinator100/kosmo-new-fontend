@kosmo/ui

Typed vanilla-TypeScript UI primitives + design tokens. Every component exposes a `create*` factory and an `enhance*` upgrader.

Components

Button Primary / pink / yellow / outline · loading + disabled states. Input Default + large variants with focus / error states. Tab bar ARIA tabs pattern with arrow-key navigation. List item Leading icon + label + trailing action; danger tone. Search Form-wrapped search with clear button. Sub-header Section divider for grouped content.
