# KOSMO Prototype

The sidebar is organized into three sections:

## Prototype

A click-through of all pages with transitions and interactivity. Content changes happen inside the plugin container — no page navigation.

## Pages

Templates of each page with states. These are standalone views showing a single page's layout and visual variations.

## Components

Individual UI elements used in the pages. Isolated previews of buttons, inputs, and other building blocks with their different states (default, hover, pressed, loading, error, etc.).
